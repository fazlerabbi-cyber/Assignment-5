What is event delegation in the context of the dom?

Event delegation is a technique in JavaScript that allows you to handle events efficiently by assigning a single event listener to a parent element, which will catch all the events triggered by its child elements. Instead of adding individual event listeners to multiple child elements, event delegation leverages event bubbling, where events triggered on child elements propagate up to their parent elements.
